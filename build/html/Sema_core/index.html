Fonctionnalités de Sema Core

Cette section décrit les principales fonctionnalités de Sema Core et explique comment les utiliser étape par étape.

Chaque module est présenté avec :

  • Son objectif ;

  • Les actions disponibles ;

  • Les étapes d’utilisation ;

  • Les points de contrôle ;

  • Les emplacements prévus pour insérer les captures d’écran.

Vue d’ensemble des modules

Parcours recommandé pour un nouvel utilisateur

1. Commencer par le Tableau de bord Sema Core pour comprendre l’activité.

2. Ouvrir le Chat et conversations pour consulter les conversations.

3. Lire le Scenario Builder avant de modifier un scénario.

4. Consulter le Flow Builder pour les parcours structurés.

Bon à savoir

Astuce

Avant toute modification importante, prenez une capture de l’état actuel et notez le nom du scénario ou du paramètre que vous allez modifier. Cela facilite les vérifications avec l’équipe.
