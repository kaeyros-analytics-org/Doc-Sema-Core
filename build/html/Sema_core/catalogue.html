Catalogue et produits

Le Catalogue permet d’organiser les produits ou services qui peuvent être présentés dans les conversations, scénarios ou catalogues interactifs.

Objectif du module

Le catalogue permet de :

  • créer ou consulter des produits ;

  • organiser les produits par catégorie ;

  • préparer les éléments affichés dans un chatbot ;

  • utiliser les produits dans le Scenario Builder ;

  • faciliter la prise de commande ou la recommandation.

Capture à insérer — Catalogue

Étape 1 — Ouvrir le catalogue

  1. Dans le menu, cliquez sur Catalogue, Produit ou Product selon le libellé affiché.

  2. Consultez la liste des produits.

  3. Vérifiez les catégories disponibles.

Capture à insérer — Liste produits

Étape 2 — Ajouter un produit

  1. Cliquez sur Ajouter un produit.

  2. Renseignez le nom.

  3. Ajoutez une description.

  4. Ajoutez le prix si le champ existe.

  5. Ajoutez une image produit.

  6. Sélectionnez une catégorie.

  7. Enregistrez.

  8. Vérifiez que le produit apparaît dans la liste.

Capture à insérer — Ajout produit

Étape 3 — Créer ou modifier une catégorie

  1. Ouvrez la gestion des catégories.

  2. Cliquez sur Ajouter catégorie.

  3. Renseignez le nom de la catégorie.

  4. Ajoutez une description si nécessaire.

  5. Enregistrez.

  6. Associez les produits à cette catégorie.

Capture à insérer — Catégories

Étape 4 — Utiliser le catalogue dans un scénario

  1. Ouvrez le Scenario Builder.

  2. Ajoutez un nœud Catalogue, Catalog Interactive ou Produit unique.

  3. Sélectionnez les produits ou catégories à afficher.

  4. Enregistrez le choix de l’utilisateur dans une variable.

  5. Connectez la suite du parcours.

Capture à insérer — Catalogue dans Scenario Builder

Bonnes pratiques

  1. Utilisez des noms courts et compréhensibles.

  2. Ajoutez des images claires.

  3. Vérifiez les prix avant publication.

  4. Supprimez ou archivez les produits indisponibles.

  5. Testez l’affichage dans une conversation avant utilisation réelle.
