Messages en masse

Le module Messages en masse permet d’envoyer une communication à plusieurs contacts en une seule opération. Il est utile pour les promotions, annonces, rappels, événements ou informations importantes.

Objectif du module

Ce module permet de :

  • Créer une campagne ;

  • Sélectionner une audience ;

  • Préparer un message ou un modèle ;

  • Envoyer immédiatement ou planifier ;

  • Suivre les résultats d’envoi.

Capture à insérer — Liste des campagnes

Étape 1 — Créer une campagne

1. Ouvrez Messages en masse. 2. Cliquez sur Créer, Nouvelle campagne ou l’action équivalente. 3. Donnez un nom explicite à la campagne. 4. Sélectionnez le canal d’envoi si l’option existe. 5. Enregistrez ou passez à l’étape suivante.

Capture à insérer — Création campagne

Étape 2 — Sélectionner les destinataires

1. Choisissez tous les contacts ou un segment précis. 2. Utilisez les filtres si nécessaire. 3. Vérifiez le nombre de destinataires. 4. Retirez les contacts qui ne doivent pas recevoir le message.

Capture à insérer — Sélection destinataires

Étape 3 — Préparer le message

Selon la configuration, vous pouvez utiliser :

  • Un texte simple ;

  • Un modèle validé ;

  • Un message avec variables ;

  • Un message avec média ;

  • Un bouton ou une redirection.

Pour préparer le contenu :

1. Rédigez le message. 2. Insérez les variables si besoin. 3. Vérifiez les caractères spéciaux. 4. Prévisualisez le rendu. 5. Validez le contenu avec l’équipe avant envoi.

Capture à insérer — Contenu du message

Étape 4 — Envoyer ou planifier

1. Choisissez Envoyer maintenant ou Planifier si l’option existe. 2. Pour une planification, choisissez la date et l’heure. 3. Vérifiez le récapitulatif. 4. Confirmez l’envoi. 5. Attendez le retour de statut.

Capture à insérer — Confirmation envoi

Étape 5 — Suivre les résultats

Après l’envoi, consultez les indicateurs disponibles :

  • nombre de messages envoyés ;

  • nombre de messages échoués ;

  • statut de livraison ;

  • réponses reçues ;

  • interactions générées.

Capture à insérer — Résultats campagne

Bonnes pratiques

1. Envoyez uniquement aux contacts autorisés. 2. Préférez un message court et direct. 3. Testez le message sur un petit groupe avant un envoi large. 4. Vérifiez les variables avant envoi. 5. Évitez les envois répétés trop rapprochés.
